:::note

This article describes the basics of collection creation and product adding. For detailed information please refer to **[Collections](../topics/products/collections/collections.mdx).**

:::

A collection represents a group of products. For example, you can create a collection for:
* seasonal products
* products on sale
* products of a certain type (t-shirts, jeans)
* products for specific customer group (men, women, children)

## Create collection {#create-collection}

<Tabs
  groupId="view-mode"
  defaultValue="desktop"
  values={[
    {label: 'Desktop', value: 'desktop'},
    {label: 'Tablet', value: 'tablet'},
    {label: 'Mobile', value: 'mobile'},
  ]}>

<TabItem value="desktop">

1. In admin interface, go to **Products** > **Collections**
2. Select **Create Collection**
3. Fill in collection **Title** and **Description**

:::tip

Learn about other collection parameters: **[Collections](../topics/products/collections/collections.mdx#collection-parameters)**.

:::

4. Select **Save**

Congratulations, you have created your collection!

5. In the pop-up window, select **Add Products**

</TabItem>

<TabItem value="tablet">

1. In admin interface, select <Image src="/img/hamburger-menu-button.png" alt="Menu button" width={20} height={20} /> and go to **Products**
2. In the sidebar, select **Collections**
3. In **Collections**, select <Image src="/img/create-object-icon.png" alt="Create collection button" width={20} height={20} />
4. Fill in collection **Title** and **Description**

:::tip

Learn about other collection parameters: **[Collections](../topics/products/collections/collections.mdx#collection-parameters)**.

:::

5. Select **Save**

Congratulations, you have created your collection!

6. In the pop-up window, select **Add Products**

</TabItem>

<TabItem value="mobile">

1. In admin interface, select <Image src="/img/hamburger-menu-button.png" alt="Menu button" width={20} height={20} /> and go to **Products**
2. In the sidebar, select **Collections**
3. In **Collections**, select <Image src="/img/create-object-icon.png" alt="Create collection button" width={20} height={20} />
4. Fill in collection **Title** and **Description**

:::tip

Learn about other collection parameters: **[Collections](../topics/products/collections/collections.mdx#collection-parameters)**.

:::

5. Select **Save**

Congratulations, you have created your collection!

6. In the pop-up window, select **Add Products**

</TabItem>
</Tabs>

## Add products {#add-products}

:::caution 

You must [create products](products.mdx#create-your-first-product) before adding them to the collection.

:::

:::note

All product changes (adding new products or deleting products from the collection) made in **Products** section are saved automatically. You do not have to save the collection if you work with its products only. 

:::

<Tabs
  groupId="view-mode"
  defaultValue="desktop"
  values={[
    {label: 'Desktop', value: 'desktop'},
    {label: 'Tablet', value: 'tablet'},
    {label: 'Mobile', value: 'mobile'},
  ]}>

<TabItem value="desktop">

1. In **Edit collection**, select **Add Product** in **Products** section
2. Mark checkboxes for the products you want to add to the collection

:::tip

To select all products at once, mark the highest checkbox.

:::

3. Select **Add**

</TabItem>

<TabItem value="tablet">

1. In **Edit collection**, select **Add Product**  in **Products** section
2. Mark checkboxes for the products you want to add to the collection

:::tip

To select all products at once, mark the highest checkbox.

:::

3. Select **Add**

</TabItem>

<TabItem value="mobile">

1. In **Edit collection**, select <Image src="/img/create-object-icon.png" alt="Add product button" width={20} height={20} /> in **Products** section
2. Select the products you want to add to the collection
3. Select **Add**

</TabItem>
</Tabs>

Now you can close **Edit collection** by selecting **Cancel** below.

:::tip

Additionally, learn [how to manage collection products](../topics/products/collections/collections.mdx#manage-products).

:::
