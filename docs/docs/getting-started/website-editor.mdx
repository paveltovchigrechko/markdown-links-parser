#### Header {#header}

:::note

In default theme, the header section settings apply to all pages. Therefore, once you set up the header for one page (for example, for the home page), other pages (product and collection pages) will render the header with the same settings.

:::

The **Header** is the topmost section on the page.

The **Header** section contains **Logo**, **Links**, and  **Announcement bar** settings.

The **Logo** settings define the image that is rendered on the left side of the header.

**Image**. You can add the logo image in three ways:
  * Select the frame and choose the file in the file explorer
  * Drag and drop the image file on the frame (for desktop version)
  * Select **Image gallery**, choose the image, and again **Select**

**Logo name**. This text appears if the logo image can't be rendered for some reason

The **Links** settings allow you to change the navigation menu used in the header. By default, the **Links** settings contain the **Header** navigation menu. 

To edit the navigation menu, select <Image src="/img/settings-button-2.png" alt="Settings button" width={20} height={20} /> in **Navigation menu**. Refer to the [Navigation article](../topics/website-editor/navigation.mdx) for more information.

The **Announcement bar** settings define the topmost bar that appears above the header. The announcement bar works as a link. You can use this bar to bring the attention of your customers to an announcement. For example, you can inform the customers about seasonal sales, new products or collections, etc.

**Show announcement**. Mark the checkbox to enable the announcement bar

**Home page only**. Mark the checkbox to restrict the announcement to the home page only. Otherwise, the announcement will appear on all website pages

**Text**. Fill in the text that will appear in the announcement bar.

:::tip

You can add emojis to the announcement text!

:::

**Link**. Fill in the URL of the announcement. After your customer selects the announcement, the customer will be redirected to this URL

**Background**. Select the color picker to set the announcement background color

**Text**. Select the color picker to set the announcement text color

:::tip

To return to the **Page editor**:
* for desktop mode use <Image src="/img/website-editor-return-button.png" alt="Return button" width={20} height={20} /> in the sidebar
* for tablet and mobile modes use <Image src="/img/website-editor-return-button.png" alt="Return button" width={20} height={20} /> in the upper panel

:::

#### Slideshow {#slideshow}

**Slideshow** is a section that contains up to 16 slides with images. You can set images, text, and interactive buttons to the slides so your customers can get additional information about your products. You can set auto-rotation for slides or let your customers rotate them manually.

The **Slideshow** section in the sidebar contains the slides list (learn [how to manage slides](#manage-slides)) and general settings:
* **Slide height**. Select the height of the slides in the drop-down menu
* **Text size**. Set the size of the text that you add to the slides in the drop-down menu
* **Text alignment**. Set the alignment of the text that you add to the slides in the drop-down menu
* **Blur image**. Mark the checkbox to enable the blurring of the slide image
* **Auto-rotate slides**. Mark the checkbox to enable the slides' auto-rotation on the website. If the checkbox is not marked, your customers have to rotate the slides manually
* **Rotation time, sec**. Move the slider to set up the time of the slides' auto-rotation

##### Manage slides {#manage-slides}

The slides in the **Slideshow** section are displayed in the sidebar. The slides appear in the **Slideshow** section of the website as they appear in the sidebar: the topmost slide in the sidebar appears as the first one on the website, and so on.

To add a new slide:

1. Select **Add slide**. The settings of the new slide appear in the sidebar
2. Set settings for the new slide (learn more in [**Slide settings**](#slide-settings))

To edit slide settings:
1. Select the slide panel. The settings of the slide appear in the sidebar
2. Change settings for the slide (learn more in [**Slide settings**](#slide-settings))

To change a slide position in the slideshow:
1. In the slide panel, select and hold <Image src="/img/drag-button.png" alt="Drag button" width={20} height={20} />
2. Move the slide in the list and release it

To delete a slide:
1. In the slide panel, select <Image src="/img/delete-button.png" alt="Delete button" width={20} height={20} />
2. In the pop-up window, select **Delete**

Alternatively, to delete a slide:

1. Select a slide
2. In the upper panel, select <Image src="/img/settings-button.png" alt="Settings button" width={20} height={20} />
3. Select **Delete**
4. In the pop-up window, select **Delete**

To return to the **Slideshow** section:
* for desktop mode use <Image src="/img/website-editor-return-button.png" alt="Return button" width={20} height={20} /> in the sidebar
* for tablet and mobile modes use <Image src="/img/website-editor-return-button.png" alt="Return button" width={20} height={20} /> in the upper panel

##### Slide settings {#slide-settings}

The slide settings are organized as groups.

**Image** settings. You can add the slide image in different ways:
* Select the frame and choose the image file in the file explorer
* Drag and drop the image file on the frame (for desktop version)
* Select **Image library**, choose the image, and again **Select**

**Text** settings contain all text related settings:
* **Heading**. Fill in the text heading
* **Subheading**. Fill in the text that appears on the slide
* **Text**. Select the text color in the color picker

:::note

The text position on the slide is defined in the Slideshow section settings.

:::

**Button** settings contain all button related settings:
* **Button text**. Fill in the button text
* **Button link**. Fill in the button link. After your customers select the button, they will be redirected to this link. Format: `#product-page`
* **Text**. Select the color of the button text in the color picker
* **Background**. Select the button color in the color picker

:::note

By default, the button background is transparent. If you change the background color, you cannot set it transparent manually. Reload the page to return to the default settings.

:::

#### Grid products {#grid-products}

**Grid products** is a section that displays up to 16 products as a tile grid. You can select the products to display and customize the grid. Each title contains the product image and parameters and serves as a link to the product page.

The **Grid products** section contains the products list (learn [how to manage products](#manage-products)) and general settings.

**Settings** contain the section-related parameters:
* **Headline**. Fill in the section headline. The headline appears above the grid
* **Number of products in a row**. Move the slider to set the number of products in a row

**Styles** settings contain the parameters related to the product tile in the grid:
* **Heading**. Select the color of the product title in the color picker
* **Subheading**. Select the color of the product description in the color picker
* **Price**. Select the color of the product actual price in the color picker
* **Hover**. Select the color of the frame of the product tile in the color picker. The frame appears upon hovering over the tile

##### Manage products {#manage-products}

The products in **Grid products** section are displayed in the sidebar. The products appear in the **Grid products** section of the website as they appear in the sidebar: the topmost product in the sidebar appears as the first to the left one on the website, and so on.

:::note

The **Grid products** displays the first image for each product. You can [edit the product](../topics/products/products.mdx#edit-product) to change its first image.

:::

To add a new product:

1. In the sidebar, select **Add product**. The field for the slide product will appear in the sidebar
2. Select the field and choose the existing product to display in the section

Alternatively, you can [create a new product](products.mdx#create-your-first-product) by selecting the **Create product** line in the drop-down menu. Also, you can edit a selected product by selecting <Image src="/img/settings-button-2.png" alt="Settings button" width={20} height={20} />.

To change a product in a grid tile:
1. Select the product panel in the sidebar. The drop-down menu for the product appears in the sidebar
2. Select a new product for the tile

To change a product tile position in the grid:
1. In the product panel, select and hold <Image src="/img/drag-button.png" alt="Drag button" width={20} height={20} />
2. Move the panel in the list and release <Image src="/img/drag-button.png" alt="Drag button" width={20} height={20} />

To delete a product tile:
1. In the product panel, select <Image src="/img/delete-icon.png" alt="Delete button" width={20} height={20} />
2. In the pop-up window, select **Delete**

Alternatively, to delete a product tile:
1. Select a product tile
2. In the upper panel, select <Image src="/img/settings-button.png" alt="Settings button" width={20} height={20} />
3. Select **Delete**
4. In the pop-up window, select **Delete**

To return to the **Grid products** section:
* for desktop mode use <Image src="/img/website-editor-return-button.png" alt="Return button" width={20} height={20} /> in the sidebar
* for tablet and mobile modes use <Image src="/img/website-editor-return-button.png" alt="Return button" width={20} height={20} /> in the upper panel

#### Footer {#footer}

:::note

In default theme, the footer section settings apply to all pages. Therefore, once you set up the footer for one page (for example, for the home page, other pages (product and collection pages) will render the footer with the same settings.

:::

The footer is the lowermost section on the page.

The **Footer** section in the sidebar contains **Links** settings and **Show payment icons** checkbox.

The **Links** settings allow you to change the navigation menu used in the section. By default, the **Links** settings contain the **Footer** navigation menu.

To edit the navigation menu, select <Image src="/img/settings-button-2.png" alt="Settings button" width={20} height={20} /> in **Navigation menu**.

To change the navigation menu:

1. In **Navigation menu**, delete the name of the current navigation menu
2. In **Navigation menu** drop-down list, select the new navigation menu

Mark the checkbox to display the icons of the payment methods available on your website.

:::note

You can set the payment methods in **[Store settings > Payments](store-settings.mdx#connect-payment-method)**.

:::

### Product page {#product-page}

When you create a product, you assign a template to it (find more in [Product article](../topics/products/products.mdx)). A template defines the layout for the product section on the website. Therefore, the product website page with one template can differ from the product website page with another template.

The drop-down menu in the **Page editor** contains all available templates for products. We recommend setting up all of them.

To start working with the **Product page**:
1. In the **Page editor** sidebar, select the topmost menu
2. Select **Product page**

:::note

If you have already set up the header and footer sections for the **Home page**, you do not need to repeat it for the **Product page**.

:::

To set up the **Product page**, you need to adjust the **Product** section.

#### Product section {#product-section}

The **Product** section is a part of the product page. It displays the product information and the button to add the product to the cart.

The **Product** settings:
* **Allow image zooming**. Mark the checkbox if you want your customers to be able to zoom the product images
* **Show comparative price**. Mark the checkbox to display the product comparative price
* **Show full description**. Mark the checkbox to display the full product description
* **Show vendor name**. Mark the checkbox to display the product [vendor](vendors.mdx) name
* **Show quantity selector**. Mark the checkbox to display the quantity selector for the product items. Your customers will be able to add multiple product items to the cart
* **Show sharing buttons**. Mark the checkbox if you want to display social nets icons for sharing the product page link

### Collection page {#collection-page}

When you create a collection, you assign a template to it (find more in [Collection article](../topics/products/collections/collections.mdx#collection-parameters)). A template defines the layout for the collection section on the website. Therefore, the collection website page with one template can differ from the collection website page with another template.

The drop-down menu in the **Page editor** contains all available templates for collections. We recommend setting up all of them.

To start working with a collection page template:
1. In the **Page editor** sidebar, select the topmost menu
2. Select **Collection page**

:::note

If you have already set up the header and footer sections for the **Home page**, you do not need to repeat it for the collection page.

:::

#### Collection section {#collection-page}

The **Collection** section is a part of the default collection page template. It displays the collection products and optionally its image and description.

The **Collection section** settings:
* **Show collection banner**. Mark the checkbox to display the collection image at the top of the section
* **Show collection description**. Mark the checkbox to display the collection description. It appears at the top of the section. If you mark **Show collection banner** checkbox, the description appears on the collection image
* **Number of products in a row**. Move the slider to set collection product number for grid row

## Website preview {#website-preview}

After you set up the website, consider checking it by selecting **Preview** in the website control bar.

The preview opens in a new browser tab. Make sure, that every page on the website appears according to your needs.

:::tip

Use the links in the preview to navigate on the website.

:::

You can always switch to the website editor and make changes. After that, just reload the tab with preview.
