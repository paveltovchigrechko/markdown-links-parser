TikTok Catalog Manager (TCM) is a part of TikTok Ads Manager. You need to have an TikTok For Business account to work in TikTok Ads Manager.
If you already have TikTok account, you can link your existing account with TikTok For Business account.

:::info

Check [TikTok documentation](https://ads.tiktok.com/help/article?aid=10001004) if you have any troubles with your account or catalog in TCM.

:::

You can configure in TCM all the places where TikTok users can find your products added to the feed.


Before using TikTok as sales channel, make sure to read [Terms of Service](https://ads.tiktok.com/i18n/official/policy/commercial-terms-of-service), [Community Guidelines](https://www.tiktok.com/community-guidelines?lang=en#31), and [Privacy and Cookie Policy](https://ads.tiktok.com/i18n/official/policy/privacy?).
