**Navigation** allows you to create and customize the website menus. You can add your menus into website sections, so your customers can use them for navigating your website pages.

:::note

You can create one-level menu only.

:::

## Menu parameters {#menu-parameters}

**Description**. Contains the menu **Name**. It is displayed in the **Navigation** sidebar.

**Items**. Contains the menu items. They represent the links displayed in the menu (find more in [Menu items](#menu-items)).

**Liquid description**. Contains the menu **Name in Liquid**. It is used as a reference for the menu in the website theme files. You need the menu Liquid name when [adding it to a website section](#add-menu).

## Menu items {#menu-items}

You can manage the menu items in **Items** section of the **Navigation** workspace.

A menu item is a single link to the store page. Each menu item has:
* **Name**. A name is an item name as it appears in the menu. It can contain 1 - 255 symbols and must be unique within the store
* **Page link**. A page link is an item URL that opens after selecting the item name 

The drop-down list in **Page** displays:
* all [active products](../products/products.mdx#status) in **Products** submenu
* all [store collections](../products/collections/collections.mdx) in **Collections** submenu
* all [published custom pages](../custom-pages/custom-pages.mdx#custom-page-parameters) in **Pages** submenu
* **+Create page ""** link for [creating a custom page](../custom-pages/custom-pages.mdx#create-custom-page)

You can also create a [custom URL](#custom-url) for a menu item. 

In **Items** section, you can:
* [Create new item](#create-item)
* [Edit item](#edit-item)
* [Create a custom URL](#custom-url) for a menu item
* [Change the items order in the menu](#change-items-order)
* [Delete item](#delete-item)

### Custom URL for menu item {#custom-url}

The custom URL in a menu item can contain up to 2048 symbols and must be unique within a store.

To create a custom URL for a menu item:
1. [Create a new menu item](#create-item) or [edit an existing one](#edit-item)
2. Optionally, fill in **Name** if you create a new menu item
3. In **Page link**, fill in your custom URL

:::tip

By default, all custom URLs are created using HTTPS. So, if your URL uses HTTPS, you can omit it in **Page link** parameter, for example:
```
custom-url.com
```

If your URL uses HTTP or another protocol, you should indicate it, for example:
```
http://custom-url.com
```

:::

4. In the **Page** drop-down menu, select **+Create page "your custom URL"**
5. Select **Save**
6. In the workspace, select **Save**

## Add navigation menu to store website {#add-menu}

To add a navigation menu to the store website, you can use:
* [**Page editor**](#page-editor). This method is available only for a limited number of sections. Your current website theme defines the sections where you can add a navigation menu in **Page editor**. For example, the default Unistore theme allows adding a navigation menu in **Header** and **Footer** sections
* [**Code editor**](#code-editor). This method requires HTML and Liquid knowledge. It allows you to make more flexible adjustments while adding a navigation menu. For example, you can add a menu exactly where you need it or output additional menu parameters other than its items

### Add navigation menu in Page editor {#page-editor}

:::info

This paragraph describes how to add a navigation menu in the default Unistore theme (Optimum). If you use another theme, refer to the theme documentation to find where and how you can add a navigation menu.

:::

To add a navigation menu in **Page editor**:
1. Access [**Page editor**](page-editor/page-editor.mdx)
2. In **Page editor** sidebar, select **Header** or **Footer** section
3. In the sidebar, find **Links** and change the navigation menu (refer to [**Header**](page-editor/optimum.mdx#links) and [**Footer**](page-editor/optimum.mdx#footer) Optimum theme documentation for the details)

### Add navigation menu in Code editor {#code-editor}

To add a navigation menu in **Code editor**, you need to understand how to insert them into the code and which parameters you can use. This paragraph provides an overview of navigation menus as code objects.

All navigation menus are stored in the global <Code>linklists</Code> object as an array.

You can access a navigation menu by using its **Name in Liquid** parameter. For example, if you have a menu with <Code>myheader</Code> Liquid name, you can access its links as: 
```
{% for link in linklists.myheader.links %}
  <!-- your code -->
{% endfor %}
```

:::caution

If your menu Liquid name has underscores or hyphens, you should use the syntax:
```
{% for link in linklists["menu-liquid-name"].links %}
  <-- your code -->
{% endfor %}
```

:::

:::tip

We recommend using navigation menu in header and footer sections.

:::

To add the navigation menu:
1. In the **Website editor**, go to **Code editor**
2. In the **Code editor**, open the section file to which you want to add the navigation menu
3. In the section file, insert the code with your navigation menu. For example:
```
{% for link in linklists["menu-liquid-name"].links %}
  <a href="{{ link.url }}">{{ link.title }}</a>
{% endfor %}
```

We recommend displaying a title and URL for each navigation link. If necessary, you can use other [link properties](#link) as well.

### Link {#link}

<p><Code>Link</Code> represents a menu item in a navigation menu.</p>

<Code>Link</Code> object properties:

* <Code>active</Code>. Boolean. Returns <Code>true</Code> if the link is active. Returns <Code>false</Code> if not. A link is active if the current URL contains or matches the link URL
* <Code>current</Code>. Boolean. Returns <Code>true</Code> if the current URL matches the link URL. Returns <Code>false</Code> if not
* <Code>handle</Code>. String. The link handle
* <Code>object</Code>. The object associated with the link
* <Code>title</Code>. String. The link title
* <Code>type</Code>. String from a value set. The link type. The property is connected with the link-associated object. Possible types:
 <ul>
  <li><Code>FRONTPAGE</Code></li>
  <li><Code>COLLECTION</Code></li>
  <li><Code>PRODUCT</Code></li>
  <li><Code>PAGE</Code></li>
  <li><Code>SEARCH</Code></li>
  <li><Code>HTTP</Code></li>
 </ul>  
* <Code>url</Code>. String. The link URL

### Linklist {#linklist}

<p><Code>Linklist</Code> object represents a navigation menu in a store. You can use it to get the navigation menu parameters and display them in a section.</p>

<Code>Linklist</Code> object properties:

* <Code>handle</Code>. String. The Liquid name for the menu
* <Code>links</Code>. Array. All menu links
* <Code>title</Code>. String. The menu name
