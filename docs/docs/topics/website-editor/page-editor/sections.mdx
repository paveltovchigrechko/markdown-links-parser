A section is a part of a website page. Sections include the content blocks that appear on the page.

This article describes general principles of working with sections.

To learn about sections in Unistore themes, refer to articles:
* [Optimum](optimum.mdx)
* [Momentum](momentum.mdx)

## Manage sections order and visibility {#order-and-visibility}

The sections appear on the website as they appear in the sidebar: the topmost section in the sidebar appears as the topmost one on the website, and so on. You can change the position of the sections. Also, you can hide a section so it does not appear on the website. 

:::note

Depending on the theme settings, changing position or hiding for some sections may not be available. 

:::

<TabItem value="desktop">

To change the position of the section:
1. In the **Page editor** sidebar, select and hold on <Image src="/img/drag-button.png" alt="Drag button" width={20} height={20} />
2. Move the section

To hide the section on the website, select <Image src="/img/visibility-button.png" alt="Visibility button" width={20} height={20} /> for the section you want to hide in the **Page editor** sidebar. The button will change to <Image src="/img/visibility-disabled-button.png" alt="Visibility disabled button" width={20} height={20} />, and the website will not render this section. 

To display the section on the website, select <Image src="/img/visibility-disabled-button.png" alt="Visibility disabled button" width={20} height={20} />.

Alternatively, to hide or to display a section:
1. In the **Page editor** sidebar, select the section
2. In the sidebar, select **Hide section** / **Show section**

</TabItem>

<TabItem value="tablet">

To change the position of the section:
1. In the **Page editor**, select <Image src="/img/hamburger-menu-button.png" alt="Menu button" width={20} height={20} />
2. In the sidebar, select and hold on <Image src="/img/drag-button.png" alt="Drag button" width={20} height={20} />
3. Move the section

To hide the section on the website:
1. In the **Page editor**, select <Image src="/img/hamburger-menu-button.png" alt="Menu button" width={20} height={20} />
2. In the sidebar, select <Image src="/img/visibility-disabled-button.png" alt="Visibility disabled button" width={20} height={20} />

The button will change to <Image src="/img/visibility-disabled-button.png" alt="Visibility disabled button" width={20} height={20} />, and the website will not render this section. 

To display the section on the website, select <Image src="/img/visibility-disabled-button.png" alt="Visibility disabled button" width={20} height={20} />.

Alternatively, to hide or to display a section:
1. In the **Page editor**, select <Image src="/img/hamburger-menu-button.png" alt="Menu button" width={20} height={20} />
2. In the sidebar, select the section
3. Close the sidebar
4. In the upper panel, select <Image src="/img/settings-button.png" alt="Settings button" width={20} height={20} />
5. Select **Hide section** / **Show section**

</TabItem>

:::
