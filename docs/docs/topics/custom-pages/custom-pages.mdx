1. In **Custom pages**, select the custom page you want to edit
2. Change the [custom page parameters](#parameters)
3. Select **Save**

## Custom page parameters {#parameters}

**Status**. Required parameter. A custom page can have *Published* or *Draft* status. The pages with *Published* status are available by the page URL to anyone on the Internet. The pages with *Draft* status are not available on the Internet.

**Page title**. Required parameter. It represents the page name and is shown at the top of the page on your store website. The title must contain 2-63 symbols. Allowed symbols: letters, digits, hyphens, underscores.

**Page content**. It is the page body. You can use all the [text editor features](#text-editor) to create a unique content for the custom page. The page content cannot exceed 100 000 symbols.

**Page template**. The page template defines the layout of the page on your store website.

**URL**. Required parameter. It is a uniform resource locator (URL) for the custom page on your store website. By default, URL is autogenerated after the **Page title** is filled in, but you can change it. URL must be unique within all custom pages of your store and must contain 2-63 symbols. Allowed symbols: letters, digits, hyphens. Example: `.../pages/contacts`

**Automatically 301 redirect if the URL changes**. This checkbox enables the redirection when you change the custom page URL. When selected, the previous page URL redirects to the new one, so your customers do not get the unknown URL error.

**Title of search snippet**. Required parameter. A search snippet is a short description of the custom page that appears in search results. The title of the search snippet appears above its description (learn more about [meta title tags](../preferences/SEO/on-page-seo.mdx#meta-title-tag)). It can contain up to 60 symbols. Any type of symbols is allowed.

**Description of search snippet**. A part of the search snippet which appears under its title (learn more about [meta description tag](../preferences/SEO/on-page-seo.mdx#meta-description-tag)). It can contain up to 160 symbols. Any type of symbols is allowed.

## Text editor {#text-editor}

The text editor allows you to customize the content of your custom page. It appears as a panel bar above the **Page content** field when you create or edit a custom page.

The text editor elements (from left to right):
1. **Heading**. It allows you to set the paragraph level. Select the element and choose the paragraph level in the drop-down menu:
    * **Text** 
    * **Heading 1**
    * **Heading 2**
    * **Heading 3**
2. **Font**. It allows you to set the font for the selected text. Select the element and choose a font in the drop-down menu
3. **Font size**. It allows you to set the font size for the selected text. Select the element and choose a font size in the drop-down menu
4. **Style**. It allows you to apply styles for the selected text. Select the element and choose a style in the drop-down menu:
    * <Image src="/img/italic-text-icon.png" alt="Italic text icon" width={22} height={22} /> for the italic text
    * <Image src="/img/bold-text-icon.png" alt="bold text icon" width={22} height={22} /> for the bold text
    * <Image src="/img/underlined-text-icon.png" alt="Underlined text icon" width={22} height={22} /> for the underlined text
    * <Image src="/img/strikethrough-text-icon.png" alt="Strikethrough text icon" width={22} height={22} /> for the strikethrough text

:::tip

You can apply several styles for the same text.

:::

5. **Alignment**. It allows you to align the current paragraph. Select the element and choose the text alignment in the drop-down menu:
    * <Image src="/img/text-alignment-left.png" alt="Text align left" width={22} height={22} /> for the left side alignment
    * <Image src="/img/text-alignment-center.png" alt="Text align center" width={22} height={22} /> for the centered alignment
    * <Image src="/img/text-alignment-right.png" alt="Text align right" width={22} height={22} /> for the right side alignment
6. **Bulleted list**. It allows you to display the current paragraph as a bulleted list item
7. **Numbered list**. It allows you to display the current paragraph as a numbered list item
8. **Line and Paragraph spacing**. It allows you to set the spacing between lines and paragraphs. <p>To set the line spacing, select the text, select the element, and select the value under **Line spacing** heading in the drop-down menu. The line spacing applies to the current and the following lines.</p><p>To select the paragraph spacing, select the element, and select the value under **Paragraph spacing** heading in the drop-down menu. The paragraph spacing applies to the current and the following paragraphs</p>
9. **Reduce indent**. It allows you to reduce the current paragraph indentation. Indent applies to the current paragraph
10. **Increase indent**. It allows you to increase the current paragraph indentation. Indent applies to the current paragraph
11. **Create link**. It allows you to create a link from the selected text. Select the element, enter the link address in the field, and select <Image src="/img/accept-icon.png" alt="Accept icon" width={22} height={22} /> on right side.<p>To delete a link:<ol><li>In the text, select the link</li><li>In the text editor, select the **Link** element</li><li>On the field's right side, select <Image src="/img/close-chip-icon.png" alt="Close icon" width={22} height={22} /></li></ol></p>
12. **Text color**. It allows you to set the color for the selected text. Select the element and choose a color in the color picker
13. **Insert table**. It allows you to insert a table. Select the element and in the drop-down menu, select **Insert table**. By default, a new table has three rows and three columns.<p>To edit a table:<ol><li>Put the input cursor inside the table</li><li>In the text editor, select **Insert table**</li><li>In the drop-down menu, select the action to perform with the table:<ul><li>**Insert table** to insert another table in the table cell</li><li>**Add column before** to add a new column to the left side of the current one</li><li>**Add column after** to add a new column to the right side of the current one</li><li>**Add row below** to add a new table row under the current one</li><li>**Add row above** to add a new table row above the current one</li><li>**Delete row** to delete the current row</li><li>**Delete column** to delete the current column</li><li>**Delete table** to delete the current table</li></ul></li></ol></p>

:::tip

You can use all the text editor elements in the table cells.

:::

14. **Insert image**. It allows you to insert an image from the URL. Select the element, insert the image URL in the field, and select <Image src="/img/accept-icon.png" alt="Accept icon" width={22} height={22} /> on the right side
15. **Insert video**. It allows you to insert embedded YouTube video. Select the element, insert the video URL in the field, and select <Image src="/img/accept-icon.png" alt="Accept icon" width={22} height={22} /> on the right side

:::caution

The text editor supports YouTube videos only.

:::
