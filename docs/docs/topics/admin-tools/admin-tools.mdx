Unistore provides you with tools to make your work in store admin interface easier. This section contains articles dedicated to such tools:

* [**Media gallery**](media-gallery.mdx). A gallery is a storage that contains media files. In Unistore, you can use galleries for products and website sections
* [**Tags**](tags.mdx). Tags are the additional marks you can assign to your products and customers for better searching and filtering
